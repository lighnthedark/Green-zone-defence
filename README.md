# Space Invaders V2

A browser-based Space Invaders game built with JavaScript, HTML5 Canvas, and Web Audio API.

## Play Now
https://space-invaders-n01i.onrender.com

## Features
- 3 difficulty levels (Easy, Normal, Hard)
- Power-ups: Health packs, Shields, Spread Shot
- Demo Mode (arcade-style attract screen)
- Theme music that speeds up as you play
- Crab-walk alien movement
- Scrolling starfield background
- Touch & Gyroscope mobile controls
